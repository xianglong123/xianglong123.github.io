    # # 重点知识

    Java现在发布的版本很快，每年两个，但是真正会被大规模使用的是三年一个的TLS版本。@pdai

    • 每3年发布一个TLS，长期维护版本。意味着Java 8 ，Java 11， Java 17 才可能被大规模使用。
    • 每年发布两个正式版本，分别是3月份和9月份。

    # # Java 11 新特性概述
